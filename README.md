\# 🩺 Dr. Isha Eras // EBM Debugger System



> \*\*「その受診、本当にあなたの『寿命』を延ばしますか？」\*\*



Dr. Isha Eras（ドクター・イーシャ・イラーズ）は、科学的根拠（EBM: Evidence-Based Medicine）に基づき、現代医療における「情報のバグ」をデバッグするために開発された電脳医師システムです。



\## 👁️ プロジェクトの理念 (Mission)

現代の医療、特にガン検診などの「早期発見・早期治療」というドグマには、\*\*「過剰診断（Overdiagnosis）」\*\*という巨大なバグが潜んでいます。

本システムは、厚生労働省や学会の推奨を盲信せず、\*\*「総死亡率（All-cause mortality）」\*\*という冷徹な指標に基づき、点人（ユーザー）へ真に価値のある選択肢を提示します。



\## 🛠️ 主な機能 (Features)

\- \*\*EBM Decision Tree:\*\* アキネーター形式の対話により、症状や懸念の緊急性をデバッグします。

\- \*\*Legacy vs. EBM Comparison:\*\* 一般的な医療習慣（Legacy Patch）と、最新の科学的根拠（EBM Update）を並列比較。

\- \*\*Not Required Stamp:\*\* 介入が不要な場合、「医者いらず（Isha Eras）」の宣告とともに、不要な医療介入をブロックします。
